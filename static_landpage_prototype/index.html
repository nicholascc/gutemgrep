About The Index

A semantic search engine over Project Gutenberg's public domain library. Every paragraph has been embedded as a vector, creating a searchable mathematical space of literary language.

Search by meaning, not keywords. Ask for "passages about the ocean at night" and discover descriptions that match the mood and content, regardless of exact phrasing.
